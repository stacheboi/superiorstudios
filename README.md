# Superior Photo Studio

## Open Locally

Open `index.html` directly in a browser, or serve the folder with any static server.

If you have Node installed:

```bash
npx serve .
```

## Edit Content

- Update gallery photos and print metadata in `gallery-data.js`.
- Update page copy and form fields in `index.html`.
- Update layout, spacing, and visual styling in `styles.css`.
- Update gallery interactions and local form preview behavior in `app.js`.

## Forms

The forms are marked up for Netlify Forms with manual invoice follow-up. If you deploy somewhere else, connect the forms to your preferred form service or endpoint.
